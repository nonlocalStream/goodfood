Use Face Recognition in Javascript
Create an Album for the Face Recognition API in the Mashape Test Console

To build our picture database for the Face Recognition API, we need to create an "album" that will hold these pictures. This is important because having a database allows the Face Recognition API to distinguish one picture from the other.

In this exercise, we will use the Mashape test console to create this album. The specific API endpoint that we need to call to do this is here - https://www.mashape.com/lambda/face-recognition#endpoint-Create-Album

Instructions

1) Click this link to go to the "Create Album" endpoint - https://www.mashape.com/lambda/face-recognition#endpoint-Create-Album

2) Think of a unique album name and put in the "album" parameter. Hit "Test Endpoint" to call the endpoint, passing in the album name we put in.

3) It will return a response, including an "albumkey" value.

4) Now that you have "album" and "albumkey", paste them to respectively to the placeholders in Line 32 and Line 33. These will be sent along as parameters when we do the call to the face recognition endpoint later.
